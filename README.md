# 亲声伴读 VoiceCare - 老人端页面

## 📱 已实现的页面

### 1. 首页 (`elder-home.html`)
**功能：** 显示子女发来的新问候 + 自由学习入口

**设计亮点：**
- 🟠 温馨橙色主题 (#FF9F43)
- 👤 超大头像 + 通知徽章（显示未读数量）
- ✨ 呼吸光晕动画效果
- 🎯 72px+ 超大触控按钮
- 🆕 **自由学习入口** - 蓝色按钮，学习自己想学的内容
- 📱 底部导航栏（归档/交流/家人）

**核心元素：**
```
- 子女头像（带脉冲光晕）
- 通知徽章（3 个新问候）
- 欢迎文案："[小明] 给您发来了新问候"
- 巨型橙色按钮："开启问候"
- 🆕 蓝色按钮："自由学习" - 想学什么自己选
```

---

### 2. 学习页 (`elder-learning.html`)
**功能：** 单词学习界面

**设计亮点：**
- 🔤 **超大字体** - 单词 8rem+（不戴老花镜也能看清）
- 🎵 录音波形可视化
- 🎧 播放时文字跳动动画
- 🔊 中文谐音开关（可切换显示）
- 📊 进度指示器（单词 1 of 3）

**核心元素：**
```
- 巨型单词显示：Passport (8rem)
- 音标：/ˈpɑːspɔːt/
- 播放按钮："听听孩子怎么说"
- 波形动画（播放时同步跳动）
- 中文含义卡片（护照/帕死跑特）
- 英文例句卡片
```

**交互：**
- 点击播放按钮 → 播放录音 + 波形动画 + 文字跳动
- 播放结束 → 自动跳转到反馈页
- 谐音开关 → 显示/隐藏中文谐音

---

### 3. 反馈页 (`elder-feedback.html`)
**功能：** 情感闭环 - 三种回应方式

**设计亮点：**
- 🌸 **送朵花** - 全屏撒花动画 + 心跳效果
- 🎤 **说一句** - 长按录音 + 录音状态可视化
- 🔁 **再听一遍** - 返回学习页

**核心元素：**
```
- 成功图标（带弹跳动画）
- 三个巨型按钮（72px+ 高度）：
  ├─ 送朵花（红色渐变）
  ├─ 说一句（蓝色渐变，长按录音）
  └─ 再听一遍（绿色渐变）
- 录音状态指示器
- 撒花庆祝动画
```

**交互：**
- 送朵花 → 全屏撒花 + 心跳图标 + 成功提示
- 按住说一句 → 录音动画 + 脉冲效果 → 松开发送
- 再听一遍 → 返回学习页

---

### 4. 自由学习页 (`elder-freestyle.html`) 🆕
**功能：** 老人自主选择学习内容

**设计亮点：**
- 📚 **6 大学习类别** - 彩色卡片区分
- 🔍 **搜索功能** - 快速查找单词
- 📖 **最近学习** - 一键复习
- 🎨 **多彩配色** - 每个类别独特颜色

**核心元素：**
```
类别卡片 (160px+ 高度):
├─ 🛫 出行旅游 (红色) - 12 个单词
├─ 🛒 超市购物 (青色) - 15 个单词
├─ 🏥 医疗健康 (绿色) - 10 个单词
├─ 🏠 日常生活 (黄色) - 18 个单词
├─ 🍽️ 餐厅就餐 (橙色) - 14 个单词
└─ 🚨 紧急情况 (深红) - 8 个单词

最近学习列表:
- 显示已学单词
- 快速复习按钮

搜索框:
- 大字体输入框
- 搜索按钮
```

**交互：**
- 点击类别卡片 → 进入该类别单词列表
- 点击最近学习播放按钮 → 快速复习
- 输入搜索 → 查找想学的单词

## 🎨 设计规范

### 颜色系统
```css
// Elder UI 温暖配色
elder-bg: #FFF8F0        // 米黄背景
warm-orange: #FF9F43     // 主色调（温馨橙）
warm-orange-dark: #E67E22 // 深橙色

// 功能色
flower-red: #FF6B6B      // 送花（红色）
reply-blue: #4ECDC4      // 录音（蓝色）
replay-green: #95E1A3    // 重听（绿色）

// 学习类别色
category-travel: #FF6B6B     // 出行旅游
category-shopping: #4ECDC4   // 超市购物
category-health: #95E1A3     // 医疗健康
category-daily: #FFD93D      // 日常生活
category-food: #FF9F43       // 餐厅就餐
category-emergency: #E63946  // 紧急情况
```

### 字体规范
```css
font-family: 'Noto Sans SC', sans-serif
// 字号
- 单词显示：6rem+ (giant)
- 标题：3.5-4.5rem (huge/massive)
- 按钮文字：2-2.5rem
- 正文：1.25-1.5rem
- 类别卡片：2rem
```

### 触控区域
```css
- 主要按钮：72px+ 高度（老人友好）
- 类别卡片：160px+ 高度
- 次要按钮：44px 高度
- 圆角：2-2.5rem（柔和感）
```

### 阴影系统
```css
.elder-shadow {
  box-shadow: 0 20px 40px -10px rgba(255, 159, 67, 0.3);
}

.halo-glow {
  box-shadow: 0 0 60px rgba(255, 159, 67, 0.4);
}
```

## 🚀 如何使用

### 本地测试
```bash
cd /home/admin/.openclaw/workspace/voicecare
python3 -m http.server 8080
# 访问 http://localhost:8080/
```

### 页面流程
```
elder-home.html
├── 点击"开启问候" → elder-learning.html → elder-feedback.html
└── 点击"自由学习" → elder-freestyle.html
                        ├── 选择类别 → 单词列表 → elder-learning.html
                        ├── 最近学习 → 快速复习
                        └── 搜索单词 → 搜索结果
```

## 🎯 设计原则

1. **消除技术恐惧** - 无输入框、无复杂导航
2. **超大触控区** - 72px+ 按钮，防误触
3. **温暖视觉** - 橙色系，非冷色调 SaaS 风
4. **即时反馈** - 动画、震动、提示音
5. **单页流** - 减少页面跳转，线性流程
6. **自主选择** - 自由学习，满足个性化需求
